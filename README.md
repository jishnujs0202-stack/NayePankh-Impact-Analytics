# NayePankh Foundation Impact Analytics Dashboard

## Project Overview

This project presents a Power BI analytics solution developed for NayePankh Foundation.

The dashboard provides insights into:

- Donations
- Volunteer Participation
- Internship Applications
- Social Media Reach
- Event Attendance

## Dashboards

### 1. Executive Dashboard
- Total Donations
- Total Volunteers
- Total Internships
- Total Reach
- Total Attendees

### 2. Volunteer Analytics
- Volunteer Growth
- Volunteer Distribution
- City-wise Volunteer Analysis

### 3. Internship Analytics
- Application Trends
- Campaign Performance
- Internship Reach Analysis

### 4. Social Impact Analytics
- Reach by Campaign
- Event Attendance
- Social Media Performance

## Tools Used

- Power BI
- DAX
- Microsoft Excel

## Dataset Disclaimer

This project uses a representative demo dataset created for educational and demonstration purposes only.

The data does not represent actual NayePankh Foundation records.
